# MyBlogs

My physics study notes, published as a GitHub Pages blog at [Chai-Maxwell.github.io](https://Chai-Maxwell.github.io/).

## Local Development

```bash
bundle exec jekyll serve
```

## Adding New Content

1. Add a new `.md` file with YAML front matter:
   ```
   ---
   layout: default
   title: Your Title
   ---
   ```
2. Update the relevant `index.md` with a link to the new page.
3. Push to GitHub — the site rebuilds automatically.
